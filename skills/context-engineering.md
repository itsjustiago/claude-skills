# context-engineering
**Source:** https://github.com/muratcankoylan/Agent-Skills-for-Context-Engineering
**Type:** Plugin
**Install:** `claude plugin marketplace add muratcankoylan/Agent-Skills-for-Context-Engineering && claude plugin install context-engineering@context-engineering-marketplace --scope user`

## What it does
Comprehensive collection of agent skills for context engineering, multi-agent architectures, and production agent systems.

## Sub-skills
- **advanced-evaluation** — Advanced agent evaluation techniques
- **bdi-mental-states** — Belief-Desire-Intention mental state modeling
- **context-compression** — Reduce context while preserving meaning
- **context-degradation** — Handle context loss gracefully
- **context-fundamentals** — Core context management principles
- **context-optimization** — Maximize context efficiency
- **evaluation** — Agent evaluation patterns
- **filesystem-context** — Use filesystem as context store
- **hosted-agents** — Patterns for hosted/cloud agents
- **memory-systems** — Long-term and working memory architectures
- **multi-agent-patterns** — Coordination patterns for multiple agents
- **project-development** — Context-aware project development
- **tool-design** — Design tools for agent use
