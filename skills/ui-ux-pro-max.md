# ui-ux-pro-max
**Source:** https://github.com/nextlevelbuilder/ui-ux-pro-max-skill
**Type:** Plugin
**Install:** `claude plugin marketplace add nextlevelbuilder/ui-ux-pro-max-skill && claude plugin install ui-ux-pro-max@ui-ux-pro-max-skill --scope user`

## What it does
Data-driven design system generator. Analyzes your project/industry and auto-selects from 96+ color palettes, 67+ UI styles, 56 font pairings, and 100+ industry-specific UX rules.

## Sub-skills
- **ui-ux-pro-max** — Main skill, auto-generates tailored design systems based on project analysis

## Supported frameworks
React, Next.js, Vue, Svelte, SwiftUI, React Native, Flutter, Tailwind CSS
