# alirezarezvani-claude-skills
**Source:** https://github.com/alirezarezvani/claude-skills
**Type:** Plugin marketplace
**Install:** `claude plugin marketplace add alirezarezvani/claude-skills` then install individual plugins

## Installed plugins
- **engineering-skills** — Core software engineering principles and patterns
- **engineering-advanced-skills** — Senior-level architecture, scalability, system design
- **fullstack-engineer** — Full-stack development with modern frameworks
- **skill-security-auditor** — Code security audit before shipping
- **docker-development** — Containerization, Docker Compose, multi-stage builds
- **aws-architect** — AWS infrastructure design and best practices
- **product-manager** — Product thinking, PRDs, feature prioritization
