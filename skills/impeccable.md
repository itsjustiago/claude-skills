# impeccable
**Source:** https://github.com/pbakaus/impeccable
**Type:** Plugin
**Install:** `claude plugin marketplace add pbakaus/impeccable && claude plugin install impeccable@impeccable --scope user`

## What it does
Design audit and refinement plugin by Paul Bakaus. Detects and fixes 24 design anti-patterns with 18 steering commands. Works across Claude Code, Cursor, Gemini CLI, Codex CLI and others.

## Commands (sub-skills)
- **/audit** — Full design audit, flags all problems
- **/critique** — Honest feedback on what's weak
- **/polish** — Fixes spacing, padding, touch targets
- **/animate** — Adds subtle, intentional motion
- **/colorize** — Improves color usage
- **/bolder** — Makes design more confident and strong
- **/delight** — Adds premium small details
- **/optimize** — Performance and visual efficiency
- **/layout** — Fixes layout and grid issues
- **/typeset** — Improves typography
- **/shape** — Refines shapes and borders
- **/adapt** — Responsive/adaptive improvements
- **/clarify** — Improves clarity and readability
- **/distill** — Strips down to essentials
- **/harden** — Makes design more robust
- **/quieter** — Reduces visual noise
- **/overdrive** — Pushes design to the extreme
- **/impeccable** — Runs full refinement suite
