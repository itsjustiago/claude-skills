# frontend-design
**Source:** https://github.com/anthropics/claude-code (official)
**Type:** Plugin
**Install:** `claude plugin marketplace add anthropics/claude-code && claude plugin install frontend-design@claude-code-plugins --scope user`

## What it does
Anthropic's official design skill. Philosophy-first approach — before writing any code, makes Claude think through purpose, tone, constraints, and differentiation. Bans generic fonts (Inter, Roboto, Arial, Space Grotesk) and predictable patterns (purple gradients, cookie-cutter components).

## Sub-skills
- **frontend-design** — Main skill, activates design philosophy before every UI task
