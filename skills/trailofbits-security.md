# trailofbits-security
**Source:** https://github.com/trailofbits/skills
**Type:** Plugin marketplace (professional security research firm)
**Install:** `claude plugin marketplace add trailofbits/skills` then install individual plugins

## Installed plugins
- **supply-chain-risk-auditor** — Scans dependencies for known vulnerabilities and supply chain risks
- **static-analysis** — Deep static code analysis for bugs and security issues
- **variant-analysis** — Finds similar vulnerabilities across your entire codebase
- **semgrep-rule-creator** — Creates custom Semgrep rules for your codebase patterns
- **second-opinion** — Gets a second security review on critical code
- **insecure-defaults** — Catches dangerous default configurations
- **mutation-testing** — Tests your tests by introducing bugs to verify they catch them
- **property-based-testing** — Generates edge case inputs automatically to find bugs
- **audit-context-building** — Builds full context before a security audit

## When to use
- Before shipping any feature that handles user data, auth, or payments
- *"Run a supply chain audit"* — checks all dependencies
- *"Security review the auth module"* — deep code review
- *"Check for insecure defaults in this config"*
