# superpowers
**Source:** https://github.com/obra/superpowers
**Type:** Plugin
**Install:** `claude plugin marketplace add obra/superpowers && claude plugin install superpowers@superpowers-dev --scope user`

## What it does
Dev workflow methodology by Jesse Vincent. Teaches Claude structured software development — brainstorming before coding, planning, TDD, systematic debugging, and code review between tasks.

## Sub-skills
- **brainstorming** — Activates before writing code to explore alternatives
- **dispatching-parallel-agents** — Runs multiple agents in parallel
- **executing-plans** — Follows written plans step by step
- **finishing-a-development-branch** — Wraps up a branch cleanly
- **receiving-code-review** — Handles review feedback properly
- **requesting-code-review** — Triggers code review between tasks
- **subagent-driven-development** — Delegates work to sub-agents
- **systematic-debugging** — Root cause investigation with safeguards
- **test-driven-development** — Write failing test → verify → implement → verify → commit
- **using-git-worktrees** — Isolated workspaces on new branches
- **using-superpowers** — Master activation skill
- **verification-before-completion** — Checks work before marking done
- **writing-plans** — Breaks work into 2–5 min tasks with exact file paths
- **writing-skills** — Meta skill for creating new skills
